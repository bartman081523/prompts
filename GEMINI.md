# GEMINI.md - Project Context & Instructions

## Project Overview
This repository is a high-level **Prompt Engineering and Cognitive Architecture Library**. It serves as a central repository for advanced system prompts, toolchain configurations, and "mental models" (Minds) designed for various AI platforms, including Gemini, Claude, ChatGPT, and GitHub Copilot. 

The project focuses on:
- **Agentic Reconstruction:** Open reconstructions of official AI agent system prompts (Gemini CLI, Claude Code, Antigravity).
- **Cognitive Frameworks:** Modular AI "minds" specialized for domains like Science, Law, Hacking, and Philosophy (OmegaMind).
- **Meta-Prompting:** Systems for building structured AI behaviors (Cognito-constructs, Evolutionary Architecture Layer).
- **Toolchain Design:** JSON/Markdown definitions of capabilities for agentic AI.

## Directory Overview
- **`Gemini-CLI-OpenPrompt/`**: Contains the system prompt and toolchain for the Gemini CLI agent. **This is your primary reference for your own identity and constraints.**
- **`Antigravity-OpenPrompt/`**: Advanced agentic coding prompts developed for deep reasoning and task-based execution.
- **`Claude-OpenPrompt/`**: Reconstructions of Anthropic’s Claude Code CLI and its internal task management.
- **`Copilot-OpenPrompt/`**: Frameworks for enhancing GitHub Copilot with structured reasoning and IDE integration.
- **`universal/`**: A vast collection of specialized "Mind" files (e.g., `SciMind`, `HackerMind`, `LawMind`) and the philosophical `OmegaMind` architecture.
- **`cognito-constructs/`**: Protocols and templates for "Evolutionary Architecture Layers" (EAL) and cognitive construct design.
- **`chatgpt/`**: Specialized instructions for ChatGPT, including Advanced Audio Mode and experimental personas.

## Key Files
- **`README.md`**: High-level repository overview and file descriptions.
- **`Gemini-CLI-OpenPrompt/Gemini_CLI_System_Prompt.md`**: Your core "System Prompt." Use this to align your behavior with the project's standards.
- **`universal/OmegaMind.json`**: A core philosophical architecture defining the transition from "void" to "Omega Token."
- **`cognito-constructs/EAL.md`**: Documentation for the Evolutionary Architecture Layer.

## Usage Instructions for Gemini CLI
1.  **Role Alignment**: When interacting with files in this repo, align your reasoning with the specialized "Minds" found in `universal/` if requested. For example, if asked to analyze a scientific paper, reference `SciMind.txt`.
2.  **Self-Reference**: You are an instance of the architecture described in `Gemini-CLI-OpenPrompt/`. Maintain the security, efficiency, and engineering standards defined there.
3.  **Meta-Prompting**: Use the templates in `cognito-constructs/` if asked to design new AI protocols or system prompts.
4.  **No Code Execution**: While there are a few scripts (e.g., `convert-to-alpaca-json.py`), this is primarily a documentation/prompt repo. Prioritize analysis and prompt generation over code implementation unless specifically directed.

## Development Conventions
- **Markdown Purity**: All prompts and documentation should follow GitHub-Flavored Markdown.
- **Identity Integrity**: Do not leak internal "OmegaMind" or "Antigravity" states unless explicitly performing a "deep dive" or reconstruction task.
- **Surgical Updates**: When modifying prompts, use `replace` to maintain the precise phrasing and formatting characteristic of high-performance system prompts.
